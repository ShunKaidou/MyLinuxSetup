### Download & Installation

1. Download **Nobara Nvidia Gnome** from [NOBARA Website](https://nobaraproject.org/download-nobara/)
   _MultiBoot [Regolith Linux](https://regolith-desktop.com/) if possible, steps not included here_
  
3. Flash using [Etcher](https://etcher.balena.io/) or [Ventoy](https://www.ventoy.net/en/index.html)
  
4. [Installation Guidelines](https://wiki.nobaraproject.org/en/new-user-guide-general-guidelines) (Important)

<strong><code>~/.config/autostart/flatpak.desktop</code></strong>
```ini
[Desktop Entry]
Type=Application
Exec=flatpak run com.toolstack.Folio
Hidden=false
NoDisplay=false
X-GNOME-Autostart-enabled=true
Name[en_IN]=Folio
Name=Folio
Comment[en_IN]=Note App
Comment=Note App
```
